# 🎨 Prompts para Meshy.ai — Assets Senta a Pua!

## Instruções
1. Acesse https://meshy.ai
2. Use o modo **Text to 3D**
3. Configure: **Style: Low Poly / Game Asset**
4. Cole cada prompt abaixo e gere
5. Baixe como `.glb` (Godot importa nativamente)
6. Salve em `assets/meshy/` no projeto
